# 保险投保自动化系统 - 项目功能说明书

## 一、项目概述

### 1.1 项目简介
本项目是一个基于Python开发的保险投保自动化处理系统，主要用于自动化处理邮件中的投保申请，解析投保信息，管理投保记录，并与保险系统对接完成投保流程。

### 1.2 技术栈
- **开发语言**: Python 3.x
- **核心库**: 
  - pandas: Excel数据处理
  - imaplib: 邮件接收
  - email: 邮件解析
  - openpyxl: Excel文件操作
  - requests: HTTP请求（预留）

### 1.3 系统架构
```
保险投保自动化系统
├── 邮件解析模块 (mail_parser.py)
├── 数据管理模块 (database.py)
├── 工具函数模块 (utils.py)
├── 配置管理模块 (config.py)
├── 记录读取模块 (read_uninsured_records.py)
└── 记录更新模块 (update_insurance_record.py)
```

## 二、核心功能模块

### 2.1 邮件解析与登记模块 (mail_parser.py)

#### 功能描述
自动连接邮箱，读取未读邮件，解析邮件内容和附件，提取投保信息并登记到Excel表格。

#### 主要功能
1. **邮箱连接**
   - 支持IMAP协议连接企业邮箱
   - 自动选择指定文件夹
   - 搜索未读邮件

2. **邮件解析**
   - 解析邮件主题、发件人、接收时间
   - 提取邮件正文内容
   - 解码多种字符编码（UTF-8、GBK、Latin-1）
   - 提取并保存邮件附件

3. **投保信息提取**
   - 自动识别投保类型（快递/非快递）
   - 提取提单号
   - 解析Excel附件中的详细投保信息
   - 分析货物类型（普货/易碎品/危险品）
   - 判断进出口类型

4. **数据登记**
   - 检查邮件是否已处理（防重复）
   - 将投保信息写入Excel记录表
   - 标记已处理邮件为已读

#### 执行时机
- 定时任务：每天12:00和21:00执行

#### 关键类
- `MailParser`: 邮件解析器主类

---

### 2.2 数据管理模块 (database.py)

#### 功能描述
管理本地Excel表格，提供投保记录的增删改查功能。

#### 主要功能
1. **Excel文件管理**
   - 按日期自动创建Excel文件（格式：投保记录_YYYYMMDD.xlsx）
   - 初始化表格结构和表头
   - 自动创建目录

2. **记录操作**
   - 添加新记录
   - 更新记录状态
   - 查询所有记录
   - 查询未投保记录
   - 按状态筛选记录
   - 检查记录是否存在

3. **数据字段**
   - 基础信息：序号、登记时间、邮件接收时间、发件人、邮件标题
   - 投保信息：投保类型、货物名称、货物类型、进出口类型
   - 状态信息：投保状态、邮件回复状态、保单号
   - 被保险人信息：被保险人、联系电话、通讯地址
   - 运输信息：起运地、目的地、运输方式、起运日期
   - 货物信息：唛头、包装数量、条款
   - 发票信息：发票号、提单号、发票金额、币种

#### 关键类
- `LocalSheetManager`: 本地Excel管理器

---

### 2.3 工具函数模块 (utils.py)

#### 功能描述
提供系统所需的各种工具函数和辅助功能。

#### 主要功能
1. **目录管理**
   - 创建目录结构
   - 按日期组织文件夹
   - 清理临时文件

2. **数据分析**
   - 分析货物类型（基于关键词匹配）
     - 危险品识别：电池、磁铁、化学品、液体、气体等
     - 易碎品识别：玻璃、陶瓷、镜子、灯具等
     - 默认：普货
   - 分析投保类型（快递/非快递）
   - 分析进出口类型（基于起运国家）

3. **数据提取**
   - 提取提单号（支持多种格式）
   - 解析Excel附件
     - 被保险人信息
     - 货物运输信息
     - 货物详细信息
     - 发票信息

4. **文件操作**
   - 保存附件到临时目录
   - 生成保单文件名
   - 文件路径处理

5. **日志管理**
   - 按应用分类记录日志
   - 按日期组织日志文件
   - 支持多种日志级别（INFO、WARNING、ERROR）

#### 配置项

   - 非快递投保系统URL
   - 登录凭证（预留）

3. **文件存储配置** (FILE_CONFIG)
   - 基础目录
   - 临时文件目录
   - 日志目录
   - Excel文件路径

4. **关键词配置** (KEYWORDS)
   - 快递关键词
   - 忽略关键词
   - 易碎品关键词
   - 危险品关键词

### 2.5 记录读取模块 (read_uninsured_records.py)

#### 功能描述
读取并显示未投保的记录，供后续投保处理使用。

#### 主要功能
1. **记录读取**
   - 读取指定Excel文件
   - 筛选投保状态为"未投保"的记录
   - 将NaN值转换为空字符串

2. **数据展示**
   - 显示未投保记录总数
   - 展示每条记录的关键信息
   - 提供字段访问示例

3. **灵活性**
   - 支持指定文件名读取
   - 支持命令行参数传入

#### 关键类
- `UninsuredRecordReader`: 未投保记录读取器

---

### 2.6 记录更新模块 (update_insurance_record.py)

#### 功能描述
更新投保记录的保单号和投保状态。

2. **数据验证**

   - 确保数据类型正确

## 三、业务流程

### 3.1 邮件处理流程
```
1. 连接邮箱 → 2. 选择文件夹 → 3. 搜索未读邮件
    ↓
4. 解析邮件内容 → 5. 提取投保信息 → 6. 保存附件
    ↓
7. 检查是否重复 → 8. 登记到Excel → 9. 标记已读
```

### 3.2 投保信息提取流程
```
邮件内容
    ↓
1. 分析投保类型（快递/非快递）
2. 提取提单号
3. 解析Excel附件
    ↓
4. 提取被保险人信息
5. 提取运输信息
6. 提取货物信息
7. 提取发票信息
    ↓
8. 分析货物类型
9. 判断进出口类型
    ↓
10. 写入Excel记录表
```

### 3.3 数据流转
```
邮件 → 邮件解析器 → Excel记录表 → 投保系统（预留）
                        ↓
                    保单文件 → 邮件回复（预留）
```

## 六、关键特性

### 6.1 智能识别
- **投保类型识别**: 基于关键词自动判断快递或非快递
- **货物类型识别**: 自动识别危险品、易碎品
- **进出口判断**: 根据起运国家自动判断

### 6.2 防重复处理
- 基于邮件接收时间、发件人、标题三要素判断
- 已处理邮件自动标记为已读
- 重复邮件跳过处理并记录日志

### 6.3 多编码支持
- 支持UTF-8、GBK、Latin-1等多种字符编码
- 自动解码邮件主题和附件名
- 确保中文内容正确显示

### 6.4 完整日志
- 按应用分类记录
- 按日期组织文件
- 支持多级别日志（INFO、WARNING、ERROR）

### 6.5 灵活配置
- 集中配置管理
- 支持多个投保系统
- 可自定义关键词规则

## 九、注意事项

### 9.1 安全性
- ⚠️ 配置文件包含敏感信息（邮箱密码），请勿提交到版本控制
- 建议使用环境变量或密钥管理服务存储凭证
- 定期更换邮箱密码

### 9.2 数据备份
- 建议定期备份Excel记录文件
- 重要附件应保留原始文件
- 日志文件定期归档

### 9.3 错误处理
- 所有关键操作都有异常捕获
- 错误信息记录到日志文件
- 失败邮件不会被标记为已读

### 9.4 性能优化
- 大量邮件处理时增加延迟（避免频繁请求）
- 定期清理临时文件
- 日志文件定期清理
